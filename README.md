# Advertisement Service API v2 (Auth & RBAC)

REST API сервис для объявлений купли/продажи с системой регистрации, авторизации, ролями и разграничением прав доступа (User/Admin).

## Особенности
- Авторизация через JWT (срок действия 48 часов).
- Хеширование паролей с использованием `bcrypt`.
- Ролевая модель доступа (RBAC).
- Полная докеризация (App + PostgreSQL).

## Требования
- Docker и Docker Compose.

## Работа с API

После запуска API доступно по адресу: http://localhost:8080

Документация Swagger: http://localhost:8080/docs

## Основные эндпоинты:

POST /login — Авторизация (возвращает JWT).

POST /user — Регистрация нового пользователя.

GET /advertisement — Поиск объявлений по title или author.

PATCH /advertisement/{id} — Редактирование (требует прав владельца или админа).

## Правила доступа

Публичные (без токена): Просмотр объявлений, поиск, регистрация пользователя.

Пользователь (User): Создание объявлений, редактирование/удаление только своих объявлений и своего профиля.

Администратор (Admin): Полный доступ к любым объявлениям и пользователям.


## Установка окружения (Локально)
Если вы хотите запускать проект не только в Docker, но и локально:
1. Создайте окружение: `python -m venv .venv`
2. Активируйте:
   - Windows: `.venv\Scripts\activate`
   - Linux/macOS: `source .venv/bin/activate`
3. Установите зависимости: `pip install -r requirements.txt`

## Запуск через Docker
1. Настройте `.env`
2. Выполните: `docker-compose up --build`
